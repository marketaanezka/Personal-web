## Co jsem se naučila
 Projekt bylo mé první seznámení s Reactem, kde jsem se ponořila do principů skládání webu z jednotlivých komponent. **Reusability, maintainability, readability.**
Jelikož produkt byl původně vyvinutý malým týmem pro interní potřeby, bylo hodně prostoru pro refaktoring, tvorbu nových komponent a podrobné code review.

<div class="mdx-space--md"></div>

Pracovali jsme ve **sprintech**, bavila mě úzká spolupráce s naší testerkou a jelikož v týmu nebyl dedikovaný UX designer, vzala jsem si na starost i návrhy nových featur. Kromě programování jsem si osvojila i základní **UX principy** a práci ve Figmě. Taky začal přepis celé aplikace z JavaScriptu do **TypeScriptu** a refaktoring css do **css modulů**. 

<div class="mdx-space--md"></div>

Poznala jsem, jak vypadá vývoj produktu naživo, jak funguje agile, code review, testování i vývoj na dev/staging/production prostředí.
